# Скрипт для добавления сетей в NDFC
## Описание

   Получает токен для работы с API NDFC.<br/>
   Позволяет создавать L2 сети в заданном диапазоне VLAN.<br/>
   Позволяет добавлять сети на коммутаторы.<br/>
   Параметры создаваемых сетей задаются в файле config.py(в директории parts).<br/>
  
# Установка и запуск:

## Клонировать репозиторий и перейти в него в командной строке:

```
git clone https://github.com/Ortega0101/NDFC.git
```

```
cd NDFC
```

## Cоздать виртуальное окружение:

Для Linux и macOS:
```
python3 -m venv env
```
Для Windows:
```
python -m venv venv
```
Активировать виртуальное окружение:
```
source env/bin/activate
```

Обновить pip:

Для Linux и macOS:

```
python3 -m pip install --upgrade pip
```
Для Windows:
```
python -m pip install --upgrade pip

```
Установить зависимости из файла requirements.txt:
```
pip install -r requirements.txt
```

## Как пользоваться скриптом:

После установки запустить на выполнение файл networks.py
и дождаться приглашения ко вводу учетных данных

При запуске без параметров, скрипт создаст сети,  выполнит attach на коммутатор и добавит подсеть на указанные интерфейсы

Усли нужно только создать сеть, запустите с параметром create

```
python networks.py --actions create
```
Усли нужно только добавить сеть на коммутатор и интерфейсы, запустите с параметром attach

```
python networks.py --actions attach
```

# Примечания

Порты коммутатора указываются строкой, через запятую, без пробела
Номера VLAN можно задать не диапазоном, а списком, через константу VLANS 

##### Пример
```python

VLANS = [10, 12, 15, 24]

```
